할인코드/프로모션 적용이 안 될 때

할인코드나 카드 프로모션이 정상 적용되지 않는 경우의 대표 사례와 해결 방법입니다.

할인코드 유효기간 만료 또는 소진

가장 흔한 원인입니다. 입력한 코드의 유효 기간이 지났거나, 선착순 수량이 모두 소진된 경우 할인 적용이 되지 않습니다.

해결 방법

새로운 유효 쿠폰을 확인하거나, KKday 이벤트 페이지에서 프로모션 진행 여부를 재확인 가능합니다. 쿠폰 업데이트는 매월 진행됩니다.

코드 사용 조건 미충족

최소 결제금액 또는 지정 카테고리 등 코드마다 사용 조건이 있습니다. 예를 들어 100,000원 이상 결제 시에만 쓸 수 있는 코드를 소액 결제에 사용하거나, 투어전용 코드를 숙박상품에 적용하려 하면 할인되지 않습니다.

해결 방법

해당 코드의 조건(최소 금액, 적용 대상 상품, 신규회원 전용 여부 등)을 확인하고, 조건에 맞게 상품/금액을 조정한 후 다시 적용합니다.

일부 상품 쿠폰 적용 불가

KKday의 몇몇 상품은 할인코드 적용이 제한되어 있습니다. 예를 들어 일부 테마파크 티켓이나 기차패스 등은 제휴사 정책상 쿠폰 적용이 안 될 수 있습니다.

해결 방법

쿠폰 적용 불가 안내가 뜨는 상품은 해당 할인코드 대상이 아니므로 다른 상품을 선택하거나, 해당 상품에 특화된 별도 프로모션이 있는지 찾아봅니다.

여러 쿠폰 동시 사용 오류

KKday는 한 번에 한 가지 할인코드만 적용됩니다. 결제페이지 쿠폰란에 두 개 이상의 코드를 넣을 수는 있지만, 가장 혜택이 큰 1개 코드만 최종 적용되고 나머지는 적용되지 않습니다.

해결 방법

할인율 또는 할인금액이 가장 큰 쿠폰 1개만 선택 적용 가능합니다. 카드 즉시할인이나 포인트 사용은 쿠폰과 중복 가능하나, 쿠폰 코드끼리는 중복 불가입니다.

결제 통화 및 수수료 문제

카드 프로모션의 경우 결제 통화를 USD로 해야 할인쿠폰이 적용되는 경우가 많습니다. 예를 들어 Visa/Mastercard 제휴 코드는 원화(KRW)로 결제하면 적용이 안 되고, 통화를 달러로 변경해야 합니다.

해결 방법
